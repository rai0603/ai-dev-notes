PROJECT 01
DiDiPo
AI 社群媒體自動發文 SaaS
大型專案

幫社群行銷人員一鍵生成 AI 文案並自動排程發佈至 Facebook、Instagram、Threads。 串接三家 AI 廠商的六個模型，支援系列文章自動生成大綱→逐篇生成配圖的完整流程。 這是我目前做過規模最大的一個 SaaS，幾乎每個主流功能都有。

技術棧
React + Vite TypeScript Tailwind Supabase shadcn/ui OpenAI API Claude API Gemini API Google Imagen 4 Meta Graph API Cloudflare Pages
完成功能
  • 6 個 AI 模型文案生成（OpenAI / Claude / Gemini）
  • AI 圖片生成 + 智能解析文案提取圖像描述
  • 系列文章：大綱→逐篇生成→排程
  • 排程日曆（可編輯、可刪除、配圖預覽）
  • Google / FB / IG / Threads OAuth 串接
  • 推薦碼系統、訂閱方案定價頁
didipo.com 開發時程：約 3 週
// 2026 Mar early → Mar 16 · ~2 週
PROJECT 02
IQ Suite
多主題智力測驗平台
中型專案

涵蓋 AI 智商、財商、情商、領導力、社群行銷力、戶外安全力共六種測驗的線上平台。 每個測驗都有完整的加權計分引擎與付費深度報告，並支援 Facebook 動態 OG 分享， 讓用戶可以分享「我的 AIIQ 超越了 88% 的受測者」這類社群卡片。

技術棧
React + Vite TypeScript Supabase Cloudflare Pages Lemon Squeezy Cloudflare Functions Python DOCX parser
完成功能
  • 6 種測驗（AIIQ / FQ / EQ / LQ / SMQ / OSQ）
  • 加權計分 + 百分位 + 人格類型判斷
  • 付費深度報告（6 份）
  • Email + Google OAuth 登入、歷史紀錄
  • 動態 OG tags 社群分享卡片
  • Python 規格書（DOCX）自動轉 TypeScript
iqai-5zv.pages.dev 開發時程：約 2 週
// 2026 Mar 18 → Mar 25 · ~1 週
PROJECT 03
零時生存 ZERO HOUR
台灣災難應變情境問答遊戲
中型專案

以台灣常見災難情境為主題的教育型問答遊戲：地震、海嘯、戰爭、異常天候、恐怖攻擊、疫情、不明生物，共 7 大情境 159 題。 玩完後有 S/A/B/C/D/F 評級與完整生存報告，付費解鎖 PDF 下載。 這是這段期間開發速度最快的專案，一週內從零到金流全部上線。

技術棧
React + Vite Zustand Framer Motion Tailwind v3 Supabase Lemon Squeezy Google Imagen 4 Cloudflare Pages
完成功能
  • 7 大情境 159 題含分支邏輯 + 評級系統
  • 動態倒數計時器（依題目字數自動調整）
  • 人物側寫問卷（9 題 + 隨機生成角色）
  • 結果分享（Facebook / IG / Threads / 下載）
  • 付費 PDF 報告（Lemon Squeezy 金流串接完成）
  • 許願池社群功能（按讚、管理員刪除）
  • 所有配圖用 Google Imagen 4 生成（90+ 張）
zerohourtw.com 開發時程：約 1 週

PROJECT 04
CitableRank · AI讚聲
AI 搜尋能見度分析工具
原型階段

台灣第一個繁體中文 GEO（Generative Engine Optimization）分析工具。 幫企業/品牌分析自己在 ChatGPT、Claude、Gemini 被推薦的狀況，提供「讚聲分數」與改善建議。 「讚聲」來自台語，意思是有人幫你說好話、幫你背書——這正是 AI 時代品牌需要的能力。

技術棧（現在）
單頁 HTML Python proxy Claude API
規劃 SaaS 版
  • 多 LLM 並測（Claude + GPT + Gemini + Perplexity）
  • 競爭對手追蹤 + 讚聲地圖
  • AI 自動生成建議問題
  • 技術棧：Next.js 14 + Supabase + Clerk + Stripe
citablerank.ai 定價：Free / NT$599 / NT$1,499 / VIP

03

這個月學到的事

⚡
AI 加速的不是打字，是決策
以前卡關是因為不知道怎麼實作。現在卡關通常是因為沒想清楚要做什麼。 Claude 幫我把「模糊需求」變成「可執行步驟」的速度，遠比我自己想快得多。
🧱
規格書寫得好，程式碼就對
每個專案開始前花 30 分鐘把功能邊界、資料結構、API 格式寫清楚， 後續 AI 生成的程式碼幾乎不需要大改。垃圾進垃圾出，這在 AI 時代更明顯。
🔁
記憶系統是長期開發的關鍵
Claude Code 的專案記憶讓我每次開新對話都不需要重新解釋架構。 這個設計讓跨天、跨週開發同一專案的成本大幅降低。
🎯
「能上線」比「夠完美」更有價值
零時生存一週從零到金流上線，關鍵是每天只問一個問題： 「今天最重要的一件事是什麼？」功能可以迭代，上線時機不能等。
04

開發心得

// 關於 AI 協作
我不是傳統意義上的全職工程師。但這個月我完成了過去可能要半年的工程量。 Claude 幫我填補的不是「不會寫程式」的缺口，而是「一個人做不完」的缺口。 它像是一個永遠在線、永遠有耐心、永遠記得你昨天說了什麼的資深工程師。
// 關於速度與品質
快速不代表粗糙。DiDiPo 有完整的 OAuth 流程、IQ Suite 有嚴謹的計分引擎、 零時生存有 159 題含分支邏輯——這些不是將就的作品。 AI 協作讓我可以在維持品質的同時，把交付速度提高 3–5 倍。
// 關於接下來
CitableRank 是這個月最後誕生的想法，也可能是最有商業潛力的一個。 GEO（AI 搜尋優化）是 SEO 之後下一個十年的戰場，台灣還沒有人在做繁體中文的工具。 這個賽道，我想認真跑。
一個月，只是開始。
